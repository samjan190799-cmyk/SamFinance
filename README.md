# SamFinance — Менеджер личных финансов для iOS

Современное iOS-приложение для удобного учета доходов и расходов, планирования бюджетов и анализа финансовых привычек. Разработано по стандартам 2026 года.

## 🚀 Технологический стек

* **iOS SDK:** iOS 17.0+
* **Язык программирования:** Swift 6.0 (со строгим режимом проверки многопоточности — Strict Concurrency)
* **Интерфейс:** SwiftUI
* **Управление состоянием:** Макрос `@Observable` (без использования устаревших `@StateObject` / `@ObservedObject`)
* **Многопоточность:** Swift Concurrency (`async/await`, `Task`, `@MainActor`)
* **Сборка проекта:** XcodeGen (`project.yml`)
* **UX/UI:** Адаптивная темная тема, плавные пружинные анимации (`.spring()`), тактильный отклик (Haptic Feedback)

## 🛠 Генерация проекта Xcode

Для генерации Xcode-проекта используется утилита **XcodeGen**. Если она не установлена, установите ее через Homebrew:

```bash
brew install xcodegen
```

Затем в корневой директории проекта выполните команду для создания `.xcodeproj`:

```bash
xcodegen generate
```

После выполнения команды в папке появится файл `SamFinance.xcodeproj`, который можно открыть в Xcode и запустить на симуляторе или реальном устройстве.

## 💎 Особенности реализации интерфейса

1. **Тактильный отклик (Haptic Feedback):** Интегрирован во все важные действия пользователя (успешное добавление транзакции, удаление, переключение вкладок).
2. **Glassmorphism & Gradients:** Карточка баланса на дашборде использует полупрозрачные размытые фоны (акрил/стекло) и стильные градиенты, подстраивающиеся под темную или светлую тему.
3. **Микро-анимации:** Добавление транзакций и интерактивный график реагируют на действия пользователя с помощью физических пружинных анимаций.
4. **Сохранение данных:** Транзакции автоматически сохраняются в формате JSON во внутренней песочнице приложения (`Documents/transactions.json`) в фоновом режиме.
